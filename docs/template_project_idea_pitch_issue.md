# 1. Project Title

Free Sewaa – Community Donation Platform

---

# 2. Problem

Many useful items such as clothes, books, furniture, and electronics are often thrown away while others struggle to afford basic necessities. People who want to donate items often do not know where to give them, and people in need may not know where to find free resources. There is a need for a simple platform that connects donors with people who need these items.

---

# 3. Target Users

The main users of this system include:

- Students
- Low-income individuals
- Community members who want to donate items
- Anyone looking for free usable items

---

# 4. What the System Will Do

Free Sewaa will provide a simple online platform where users can post items they want to donate and others can browse those items. Users who need something can request the item and contact the donor to arrange pickup. The system will make it easier for communities to share resources and reduce waste.

---

# 5. Core Features (First Version)

- Post an item for donation
- View available donated items
- Request an item
- Contact the donor
- Display item information (category, description, location)

---

# 6. Demo Scenario

Step 1  
User opens the Free Sewaa website.

Step 2  
A donor posts a donation item (example: used textbook).

Step 3  
Another user views the list of available items.

Step 4  
The user requests the item and contacts the donor.

---

# 7. Minimum Viable Product (MVP)

The MVP will be a simple website where users can post donation items and others can view the list of items available. Users will be able to request an item and contact the donor. This version will demonstrate the core idea of connecting donors with people who need items.
